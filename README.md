## Run YourQuotes-App

```bash
npm install
```

```bash
npm run dev
```

## It has the following features:

- Gradient Theme.
- Beautiful Cards for Quotes.
- Card has footer in which footer have date, download icon, share icon, clipboard icon and delete icon.
